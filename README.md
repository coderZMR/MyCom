# MyCom
## 项目介绍
MyCom是基于Qt框架、QWT组件、Qt3d组件、3D建模软件Maya，使用C++、QML、XML语言实现了具有实时摄像显示、实时数据显示、3D姿态模拟、动态数据显示、串口设置功能的水下观测型机器人的上位机软件。
